1. 登录接口

1. 登录

接口功能

易招登录

URL

/api/console/sysUser/login

支持格式

JSON

HTTP请求方式

GET

请求参数··
参数 必选 类型 说明
username ture string 请求的用户名
password true string 请求的密码
返回字段
返回字段 字段类型 说明
code int 返回结果状态。0：正常；1：错误。
id int 用户id
mobile string 用户手机
noReadMessageCount int 未读消息
orgId int 机构ID
realName string 真实名称
role string 角色
roles string[] 权限列表
score int 可用积分
skipAuth string 认证状态
userType string 用户类型
username string 用户名
msg string
接口示例

地址：http://www.api.com/index.php?name=”可口可乐”&type=1
``` javascript
{
“code”:0,
“data”:{
“id”:3, //用户ID
“mobile”:”15386386870”,//用户手机
“noReadMessageCount”:0,//未读消息
“orgId”:1,//机构ID
“realName”:”周艳平”,//真实名称
“role”:”ORG_ADMIN”,//角色
“roles”:[],//权限列表
“score”:118,//可用积分
“skipAuth”:”N”,//认证状态
“status”:1,//用户状态
“userType”:”zhaopin-sys-user”,//用户类型
“username”:”xxrc”//用户名
},
“msg”:”请求成功”
}

3. 分页获取待服务列表

3. 分页获取待服务列表

接口功能

分页获取待服务列表

URL

/api/console/serviceProcess/queryServiceIngPage

支持格式

JSON

HTTP请求方式

GET

请求参数
参数 必选 类型 说明
page true Page 分页对象
serviceStatus true Long 服务中详细状态（0 开始服务 1待面试 2面试失败 3待入职 4 入职失败 5过保失败）
type true Long 服务权状态 0 全部 1,紧急 2，即将
返回字段
返回字段 字段类型 说明
code int 返回结果状态。0：正常；1：错误。
pageNo int 当前页
pageSize int 每页记录条数
rowCount int 总条数
pageNum int 总页数
realName string 真实姓名
gender int 性别
cityName string 当前地址
surplusDay int 剩余服务天数
mobile string 手机号
id int 服务权记录ID
serviceName string 招聘专员
projectName string 项目名称
age int 年龄
status int 服务权状态
createDate int 服务权创建时间
serviceStatus int 服务中详细状态
orgId int 机构ID
status int 服务状态
msg string 描述
接口示例

地址：/api/console/serviceProcess/queryServiceIngPage
``` javascript
{
“code”:0,//响应码
“data”:{
“page”:{
“pageNo”:1,//当前页
“pageSize”:20,//每页记录条数
“rowCount”:1,//总条数
“pageNum”:1//总页数
},
“items”:[
{
“realName”:”周六”,//真实姓名
“gender”:0,//性别
“cityName”:”深圳市”,//当前地址
“surplusDay”:7,//剩余服务天数
“mobile”:”18911112222”,//手机号
“id”:12,//服务权记录ID
“serviceName”:”周专员”,//招聘专员
“projectName”:”这是项目名称”,//项目名称
“age”:58,//年龄
“status”:1,//服务权状态
“createDate”:1563775323000//服务权创建时间
}
],
“countSum”:{ //入参
“serviceStatus”:0,//服务中详细状态
“orgId”:1,//机构ID
“status”:1//服务状态
}
},
“msg”:”请求成功”
}
