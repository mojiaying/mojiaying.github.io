            # 微分概念

            微分

            设y=f(x)在点x0某领域内有定义，且x0+Δx在该邻域内，对于函数增量，若存在与Δx无关的常数A，使得： \begin{align*} &设y=f(x) 在点x_0某领域内有定义，且x_0+ \Delta x 在该邻域内，对于函数增量，\\ &若存在与\Delta x无关的常数A，使得：\\ \end{align*} ​设y=f(x)在点x0​某领域内有定义，且x0​+Δx在该邻域内，对于函数增量，若存在与Δx无关的常数A，使得：​

            Δy=f(x0+Δx)−f(x)Δy=AΔx+o(Δx) \Delta y = f(x_0 + \Delta x ) - f(x) \\ \Delta y = A\Delta x + o(\Delta x) \\ Δy=f(x0​+Δx)−f(x)Δy=AΔx+o(Δx)

            其中o(Δx)Δx→0时比Δx更高阶的无穷小，则称f(x)在x0处可微，并称AΔ为f(x)在x0出的微分，记作dy∣x=x0=AΔx或者d[f(x)]∣x=x0=AΔx，由于Δx=1∗Δx+0,于是一元函数微分学中规定，Δx=dx,故： \begin{align*} &其中 o(\Delta x) \Delta x \to 0 时比 \Delta x 更高阶的无穷小，则称f(x)在x_0处可微，\\ &并称A\Delta 为 f(x)在x_0出的微分， 记作dy\mid _{x=x_0}=A\Delta x 或者d[f(x)]\mid _{x=x_0} =A\Delta x ，\\ &由于\Delta x = 1*\Delta x + 0 , 于是一元函数微分学中规定，\Delta x = dx, 故： \end{align*} ​其中o(Δx)Δx→0时比Δx更高阶的无穷小，则称f(x)在x0​处可微，并称AΔ为f(x)在x0​出的微分，记作dy∣x=x0​​=AΔx或者d[f(x)]∣x=x0​​=AΔx，由于Δx=1∗Δx+0,于是一元函数微分学中规定，Δx=dx,故：​

            dy∣x=x0=Adx dy\mid _{x=x_0} = Adx dy∣x=x0​​=Adx
