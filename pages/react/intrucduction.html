                                        # 什么是react

                                        1. react是构建用户界面的库，与它的生态库构成一个前端框架，2013年Facebook开源
                                        2. 虚拟DOM（原声DOM节点属性方法过于复杂，操作影响性能，Object描述页面结构提性能）
                                        3. 组件化开发
                                        4. 声明式编程（直接给你想要的，关注结果）------命令式编程（命令机器怎么去做，关注步骤）
                                        5. 单向数据流（用属性和状态映射数据流，数据流由外层到内层层传递）

                                        react-dom ： 将虚拟DOM渲染成真实DOM

                                        # 组件创建

                                        React 应用程序由组件组成。组件是 UI（用户界面）的一部分，具有自己的逻辑和外观。组件可以小到一个按钮，也可以大到整个页面。

                                        React 组件是返回 DOM对象 的 JavaScript 函数

                                        1. 创建一个组件
                                        function MyButton() {
  return (
    <button>I'm a button</button>
  );
}
                                        1. 把组件嵌入到另外一个组件中：
                                        export default function MyApp() {
  return (
    <div>
      <h1>Welcome to my app</h1>
      <MyButton />
    </div>
  );
}

                                        Note

                                        1. 组件名称必须以大写字母开头，用于区别HTML标签和组件标签
                                        2. export default关键字指定文件默认导出的内容

                                        # 使用JSX编写 DOM对象

                                        JSX比HTML更严格

                                        1. 标签必须关闭
                                        2. 组件只能返回一个根标签
                                        3. 使用className来指定类名
                                        4. {}使用花括号来存放JS变量或表达式
                                        return (
  <>
  <h1>
    {user.name}
  </h1>
  <img
  className="avatar"
  src={user.imageUrl}
  alt={'Photo of ' + user.name}
  style={{
    width: user.imageSize,
    height: user.imageSize
  }}
/>

  </>
);

                                        style={{}}不是特殊语法，而是常规的对象{}在 style={}中

                                        如果有很多HTML要转化成JSX可以使用在线转换器

                                        # 条件渲染

                                        react 没有特殊的语法写条件判断，写法跟常规JS 写法一样。

                                        1. if else 写法
                                        let content;
if (isLoggedIn) {
  content = <AdminPanel />;
} else {
  content = <LoginForm />;
}
return (
  <div>
    {content}
  </div>
);
                                        1. 三元运算符写法
                                        <div>
  {isLoggedIn ? (
    <AdminPanel />
  ) : (
    <LoginForm />
  )}
</div>
                                        1. 短路运算符写法
                                        <div>
  {isLoggedIn && <AdminPanel />}
</div>

                                        # 列表渲染

                                        列表渲染可以通过for循环或数组的map()来实现

                                        const products = [
  { title: 'Cabbage', isFruit: false, id: 1 },
  { title: 'Garlic', isFruit: false, id: 2 },
  { title: 'Apple', isFruit: true, id: 3 },
];

export default function ShoppingList() {
  const listItems = products.map(product =>
    <li
      key={product.id}
      style={{
        color: product.isFruit ? 'magenta' : 'darkgreen'
      }}
    >
      {product.title}
    </li>
  );

  return (
    <ul>{listItems}</ul>
  );
}

                                        Note

                                        li具有 key属性，要传唯一标识项，稍后插入修改或排序使用

                                        # 事件响应

                                        在组件内声明事件处理来响应事件

                                        function MyButton() {
  function handleClick() {
    alert('You clicked me!');
  }

  return (
    <button onClick={handleClick}>
      Click me
    </button>
  );
}

                                        # 页面响应数据更新

                                        引入useState勾子，在组件内生命状态变量,useState传参为变量默认值，useState`变量包含当前变量和更新函数

                                        import { useState } from 'react';
function MyButton() {
 const [count, setCount] = useState(0);
 function handleClick() {
   setCount(count + 1);
 }
return (
   <button onClick={handleClick}>
     Clicked {count} times
   </button>
 );
}

// 多次使用组件，每个组件的状态便令是独立的
export default function MyApp() {
 return (
   <div>
     <h1>Counters that update separately</h1>
     <MyButton />
     <MyButton />
   </div>
 );
}

                                        勾子使用比较严格，只能组件顶部调用hook，

                                        # 组件数据共享

                                        上面的例子每个组件都有独立的state变量，如果要多个组件共享一个state一起更新数据，可以把state放在父组件，点击按钮通知副组件更新状态。

                                        export default function MyApp() {
  const [count, setCount] = useState(0);

  function handleClick() {
    setCount(count + 1);
  }

  return (
    <div>
      <h1>Counters that update together</h1>
      <MyButton count={count} onClick={handleClick} />
      <MyButton count={count} onClick={handleClick} />
    </div>
  );
}

function MyButton({ count, onClick }) {
  return (
    <button onClick={onClick}>
      Clicked {count} times
    </button>
  );
}
